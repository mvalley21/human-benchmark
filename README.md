## Built With

- ReactJs

## Getting Started

- clone the repository by running
  `git clone https://github.com/rachidelaid/human-benchmark.git`
- navigate to the folder
  `human-benchmark`
- install all the dependencies
  `npm i`
- run the app locally
  `npm run start`
